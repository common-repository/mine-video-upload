== Description ==
本插件需结合Mine Video Player使用，可直接将本地视频上传至网站，并配合主插件实现播放。

使用方法请查看 <a href="https://www.zwtt8.com/plugins/2020-12-05/mine-video-player-upload/">https://www.zwtt8.com/plugins/2020-12-05/mine-video-player-upload/</a>


== Installation ==
1、把mine-video-upload文件夹上传到/wp-content/plugins/目录下
2、在后台插件列表中激活mine video player - Aliplayer
3、在“Mine视频播放”菜单中输入相关设置信息

== Frequently Asked Questions ==
1.可以上传什么格式？
* 上传功能是调用wp的media上传功能，以wp支持的上传类型为准。
2. 可以上传多大的视频文件？
* 上传文件大小取决于php.ini中的upload_max_filesize值。
